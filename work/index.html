Work

Verizon Cloud Computing


Lowell, MA,
Summer 2014

I was a member of the hypervisor team, developing the compute backend for cloud hosting. I designed and implemented a system to automatically export the memory dumps and log files off of crashed hypervisors, onto a long-term and permanant storage platform to enable the developers and maintaners to be able to more efficiently and easily debug and recover from problems, without impacting the production environment and resources. I also helped implement a system and dev-ops tools for long term secure secret key management across releases.
